# Saving Life and Property Bank — Online Banking Portal

A premium, professional banking web application built with React + Vite.

## Features
- Multi-stage secure login (credentials + 5-digit PIN)
- Full dashboard with live clock, balance overview, charts
- Transaction history with filters
- Cards management (Debit, Credit, Travel)
- Wire transfer wizard (3-step flow)
- Investment portfolio with holdings
- Settings with security toggles
- Help center with FAQ
- Dark / Light mode toggle
- Fully responsive — desktop + mobile

## Deploy on Vercel

1. Push this folder to a GitHub repository
2. Import the repo on [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — click **Deploy**
